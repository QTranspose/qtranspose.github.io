HTB - Blue

Nov 11, 2020 2020-11-11T00:00:00+11:00 by QTranspose
Updated Feb 24 2021-02-24T01:40:39+11:00 2 min

Blue is a vulnerable virtual machine created by ch4p on HackTheBox. In this post, we document a complete walkthrough of pwning this machine.

Enumeration

Nmap

Starting off with the nmap scan, we discover that the target is running smb service. The smb-os-discovery NSE script also tells us the OS running on the target is Windows 7 SP1.
